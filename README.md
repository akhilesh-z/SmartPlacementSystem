# Smart Placement Management System

A web-based Smart Placement Management System developed using Java, JSP, Servlets, JDBC, and MySQL to automate campus recruitment activities.

## Features

### Student Module
- Student Registration
- Student Login
- Profile Management
- Resume Upload
- View Eligible Jobs
- Apply for Jobs
- Track Application Status

### Company Module
- Company Registration
- Company Login
- Post Jobs
- View Applicants
- Download Resume
- Select / Reject Students

### Admin Module
- Admin Login
- Manage Students
- Manage Companies
- Manage Jobs
- View Applications
- Dashboard
